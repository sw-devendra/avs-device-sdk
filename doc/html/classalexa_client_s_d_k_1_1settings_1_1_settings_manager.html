◆ SettingType

template<typename... SettingsT>
template<size_t index>
using alexaClientSDK::settings::SettingsManager< SettingsT >::SettingType = typename std::decay<decltype(*std::get<index>(std::tuple<SettingsT*...>()))>::type

Define the setting type kept on position index.

Explanation:

  • tuple definition: This allow us to use std::get<> to get a specific index of SettingsT.
  • Pointers: Since a setting type can be abstract, we use pointers.
  • decltype: Gets the reference type of the "object" in index position.
  • decay: Extract the type from the reference returned by decltype.
